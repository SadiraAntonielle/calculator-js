# 🧮 Calculadora Web

Calculadora responsiva inspirada no design da calculadora do iPhone, desenvolvida com foco em praticar lógica de programação, manipulação do DOM e responsividade.

O projeto realiza operações matemáticas básicas de forma rápida, com interface limpa e adaptável para diferentes dispositivos.

---

## ✨ Funcionalidades

* ✅ Adição
* ✅ Subtração
* ✅ Multiplicação
* ✅ Divisão
* ✅ Limpar display (AC)
* ✅ Alternar sinal (+/-)
* ✅ Cálculo de porcentagem (%)
* ✅ Layout responsivo (mobile, tablet e desktop)
* ✅ Interface inspirada na calculadora do iPhone

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de:

* praticar lógica de programação
* treinar manipulação do DOM
* trabalhar responsividade
* fortalecer fundamentos de JavaScript
* compor portfólio de desenvolvimento front-end
